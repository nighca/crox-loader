# [crox](https://github.com/thx/crox) loader for webpack

## Installation

`npm install crox-loader`

## Usage

``` javascript
var renderSample = require("crox!./sample.tpl");
```

[Documentation: Using loaders](http://webpack.github.io/docs/using-loaders.html)
